StereoMap 4.1.0-4.1.1 更新重点：

  • 您现在可以在细胞级分辨率下直观地探索单个或选定特征的分布。
  • 对于由SAW 8.1.0或更高版本输出的Stereo-CITE数据集，您可以在联动窗口中轻松可视化蛋白质及其对应的标记基因。
  • 从可视化Visual Explore或图像处理Image Processing导出文件时，系统会提示您选择保存位置和文件名。
  • 新的图像配准方法！通过标记特定特征点，将图像与空间特征表达矩阵对齐。这个方式无需导入矩阵，但请确保您的图像通过质量控制（QC），并且四个芯片边缘或角落可见。
  • Image QC中的图像清晰度评估已取消。
  • 为避免悬浮信息的遮挡影响 lasso 的操作，可视化Visual Explore模块在套索过程中，软件会默认关闭悬浮信息。
  • 优化可视化展示 bin1 热图的加载速度，优化后可以更加丝滑的可视化您的数据。
  • 图像处理Image Processing第二步手动配准模块使用的矩阵热图的 bin 大小从 bin20 修改为 bin10。
  • 图像处理Image Processing第五步导出手动结果时，新增配准后 TIF 图的输出路径说明。
  • 新的适配！MacOS 第一个版本发布啦。 (12月, 2024)

StereoMap 4.0.0-4.0.1 更新重点：

  • 全新的图像处理用户界面！图像QC、手动处理和可视化探索整合在一个桌面应用StereoMap中。
  • 支持通过选择一个.stereo文件来可视化数据集的所有结果。
  • 可视化Visual Explore中手动标记感兴趣区域后，可对标记区域增加标签，并对标签进行分组。
  • 可视化Visual Explore可将将差异表达分析参数导出，导出后的参数可接入SAW，并分析输出结果再次进行可视化。
  • 图像处理功能（包括图像配准、组织分割和细胞分割）已集成到StereoMap中（Image Processing），您可以按照推荐的步骤完成手动图像操作。

文件大小：216.6M

StereoMap 3.1.1-3.1.3 更新重点：

  • 支持可视化展示Stereo-CITE数据集。您可以可视化binN和细胞分辨率的空间蛋白热图，以及空间或UMAP投影的聚类结果。
  • 您可以打开一个联动窗口并排显示基因和蛋白表达图。画布通过对应的“点”关联。例如，当您在一个窗口中选择一个类群时，另一个窗口中共享相同的坐标的点也会被高亮显示，并保留其原始颜色。此外，您也可以使用套索工具来实现关联查看。
  • 现在可以通过MID过滤所选基因或蛋白的表达数据。调整后的MID范围可以导出为JSON文件，再接入到SAW中，最后可以导出过滤后的基因或蛋白的GEM矩阵文件。
  • 新增测距小工具。
  • 修复配准模板无法展示的问题。
  • 修复输出lasso JSON文件异常的问题。
