ImageStudio 2.1.2 (8月, 2023)

Windows版本点击下载
文件大小：785.1M

ImageStudio 2.1.0-2.1.2 更新重点：

  • 支持对QC生成的.tar.gz 图像直接重新QC。
  • DAPI\&mIF场景下新增适配蔡司多通道显微镜拍摄的染色图。
  • 新的拼接算法上线！新方法利用相邻FOV重叠区域的相似度信息拼接生成最终的拼接图像。
  • 支持经过QC处理后（无论QC成功或失败）的图像 .tar.gz文件直接接入图像拼接、组织分割和细胞分割模块，而无需再提前经过SAW处理。
  • 组织和细胞分割模块的矩形分割工具现在细分为两个功能：替换和更新。替换功能会把旧mask完全替换掉，同时只有选中区域更新成新的mask。相对的，更新功能会只修改选中区域的mask，保留其他区域的mask不变。

ImageStudio 2.0.2 (5月, 2023)

Windows版本点击下载
文件大小：703.0M
md5: c67e9acb33a2a1dce5e49263ade7d440

ImageStudio 2.0.0-2.0.2 更新重点：

  • 支持在一个QC任务中同时处理1张DAPI和最多6张多重免疫荧光图像。支持的IF图像类型为单通道免疫荧光图。
  • 支持DAPI图的手动拼接。
  • 新增校准模块，根据形态学特征，将IF图像和DAPI图像对其。
  • 在免疫荧光（IF）图像中，根据荧光强度分割组织。通过设定IF强度阈值来确定“组织”区域。通过调整最小和最大值，识别并标记出强度在此范围内的像素作为“组织”区域。
  • 在免疫荧光（IF）图像中，IF的细胞分割是对DAPI分割出的细胞做过滤，留下落在IF的组织区域的细胞。
  • 修复几个QC DAPI和mIF图像的问题。
  • 更新技术支持联系邮箱。

ImageStudio 1.0.0 (2月, 2023)

Windows版本点击下载
文件大小：646.9M
md5: 3bba473acdeacfa352c0a44ff0898438

ImageStudio 1.0.0 更新重点：

  • Stereo-seq图像处理桌面端软件的首个版本。包括图像QC、图像拼接、组织分割、细胞分割模块。
  • 图像拼接：标记track线交点来辅助拼接图像。
  • 组织分割：
    • 矩形套索和自动检测：这个工具支持画出矩形选区，同时ImageStudio会自动识别选区内的组织区域。适用于快速选择大范围区域。
    • 多边形套索：这个工具可让您创建不规则形状的区域进行分割。它比矩形套索更适合精确选择大区域。
    • 画笔和橡皮擦：这两个工具使您能够手动绘制区域以包括或排除它们，从而有助于精细分割小区域。
    • 腔体填充和噪声过滤：这些工具填补图像中的空隙并去除小的、不相关的细节。
  • 细胞分割：
    • 画笔和橡皮擦：这两个工具使您能够手动绘制细胞区域以包括或排除它们，从而有助于精细微调分割。
    • 自动检测：在自动识别细胞区域，在稀疏细胞区域效果较好。
